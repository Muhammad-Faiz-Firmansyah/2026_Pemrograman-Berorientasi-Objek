---

## 📚 Deskripsi

Pemrograman Berorientasi Objek (Object-Oriented Programming / OOP) merupakan paradigma pemrograman yang berfokus pada penggunaan **class** dan **object** untuk membangun program yang lebih terstruktur dan modular.

Seluruh kode dalam repository ini ditulis menggunakan **Python** dan disimpan dalam bentuk **Jupyter Notebook** untuk memudahkan proses pembelajaran, dokumentasi, dan eksperimen kode.
